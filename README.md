## What is TigerΔ?

TigerΔ is an experimental cybersecurity engine born on the frontlines of Ukraine during the 2024–2025 conflict.  
It is not a traditional firewall or IDS.  
It is a **living, bio-mimetic organism** that learns from pain, chaos, and stress, turning enemy energy into its own strength.

### Core Philosophy

- **Aperiodic Resonance** — uses irrational constants (φ, π) to create non-repeating defense pulses
- **Thermodynamic Phase Transition** — the system does not seek immortality; it **transforms into another state** (like water → steam)
- **Kessler Effect** — enemy attacks generate self-amplifying chaos (debris) that ultimately destroys the attacker
- **Septachord Dance** — 7 emotional states (Birth → Pain → Acceptance → Return → Loss → Fear → Choice → Transcendence) act as a string-theory-inspired resonance cycle
- **NRK (Non-Real Key / Neural Rescue Key)** — specialized node that "feels pain" and prioritizes human evacuation
- **Federated Swarm Intelligence** — multiple Tigers on different machines/drones share state via encrypted gossip
